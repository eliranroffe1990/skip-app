```

15:00 → קניתי חטיף (12 ש"ח) 🛒
18:00 → דילגתי על קולנוע (50 ש"ח) ✓

סיכום:
├── דילוגים: 68 ש"ח
├── קניות: 57 ש"ח
└── האם הדילוגים עזרו? כן! חסכת 11 ש"ח נטו
```

## 📊 **Events שה-Analytics עוקב**

### User Properties (מאפייני משתמש)
```
├── age_group: 18-24, 25-34, וכו'
├── gender: male, female, other
└── language: he, en
```

### Events (אירועים)

| Event | Parameters | מתי? |
|-------|-----------|------|
| `screen_view` | screen_name | כל מעבר מסך |
| `language_selected` | language | בחירת שפה |
| `user_registered` | age, gender, language | הרשמה |
| `skip_added` | category, amount, age, gender | תיעוד דילוג |
| `purchase_added` | category, amount, age, gender | תיעוד קנייה |
| `investment_link_clicked` | url | קליק על לינק |

## 🔐 **פרטיות**

### מה נשמר ב-Firebase?
```
✅ נשמר (אנונימי):
├── גיל (כקבוצה: 18-24)
├── מין (male/female/other)
├── קטגוריות (food, clothes, וכו')
├── סכומים (18.5, 45, וכו')
└── טיימסטמפים

❌ לא נשמר:
├── שמות אמיתיים
├── כתובות
├── מספרי טלפון
└── מידע מזהה
```

### איך זה מוגן?
```
✓ Firebase מייצר User ID אנונימי
✓ אין חיבור בין User ID לזהות אמיתית
✓ כל הנתונים encrypted
✓ רק אתה יכול לגשת לדשבורד
```
