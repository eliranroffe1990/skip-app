## 🔧 פתרון בעיות

### האפליקציה לא נפתחת?
- נקה את ה-cache של הדפדפן
- רענן את הדף (F5)
- נסה דפדפן אחר

### הנתונים נעלמו?
- ודא שאתה לא במצב גלישה פרטית (Incognito)
- בדוק הגדרות אחסון בדפדפן
- הנתונים נשמרים ב-localStorage

### Service Worker לא עובד?
- דרוש HTTPS (Netlify נותן אוטומטית)
- בדוק ב-DevTools → Application → Service Workers

### השפה לא משתנה?
- נקה localStorage
- רענן את הדף
- בחר שפה מחדש במסך ההרשמה
