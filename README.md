# 🚀 SKIPAPP v5.1 - חיסכון חכם

## ✨ **מה חדש בגרסה 5.1?**

### 🆕 **תכונות חדשות:**
- 🌍 **בחירת שפה ראשונה** - עברית או English
- 📋 **תנאי שימוש** - מסך הסבר מפורט
- 👤 **הרשמה פשוטה** - רק 3 שדות (שם, גיל, מגדר)
- 🔙 **כפתורי ניווט** - חזרה ובית בכל מסך
- 💾 **שמירה אוטומטית** - אין צורך ללחוץ "שמור"
- 📊 **טבלת היסטוריה משודרגת** - השוואה ויזואלית בין דילוגים לקניות
- 🎯 **קיבוץ לפי תאריך** - ראה כל הפעולות שלך ביום אחד

---

## 🎯 **אודות האפליקציה**

SKIPAPP עוזרת לך לעקוב אחרי הדילוגים שלך - הסכומים הקטנים שדילגת עליהם במהלך היום. כל קפה, חטיף, או קנייה מיותרת שלא קנית - זה כסף שחסכת!

### **איך זה עובד?**
1. 💸 תעד כל דילוג (קפה, ארוחה, קנייה)
2. 🏆 צבור נקודות (50% מהסכום)
3. 💡 למד על השקעות חכמות
4. 📊 ראה כמה חסכת עד עכשיו!
