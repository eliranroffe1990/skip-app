# SKIPAPP – גרסה נקייה (v4.7.4)

- index.html נקי ללא רצפי escape מסובכים.
- Service Worker v4.7.4 עם CACHE חדש ושבירת קאש ישן.
- תיקיות: `icon/` ו-`images/` עם קבצים תקינים.

## פריסה
1. העלה את כל הקבצים לשורש הריפו.
2. Settings → Pages → Source: Deploy from a branch → `main` / `root`.
3. פתח: `https://<user>.github.io/skip-app/`.
4. DevTools → Application → Service Workers → Unregister → רענון קשיח (Ctrl+Shift+R).
